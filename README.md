# Launch-Website-On-AWS-By-Using-Terraform

## Create .env File
vim .env

export AWS_ACCESS_KEY_ID=<Enter_your_AWS_ACCESS_KEY_ID>

export AWS_SECRET_ACCESS_KEY=<Enter_Your_AWS_SECRET_ACCESS_KEY>

export AWS_REGION=eu-north-1

## Terraform
terraform init

terraform validate

terraform plan

terraform apply
